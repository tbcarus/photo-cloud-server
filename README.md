# 📸 Photo Cloud Server

**Photo Cloud Server** — это серверная часть проекта для организации собственного облачного хранилища фотографий и других медиафайлов.  
Проект рассчитан на развёртывание на личном сервере и предоставляет REST API для регистрации пользователей, загрузки, хранения и скачивания файлов.

---

## 🚀 Возможности

- 📂 **Управление файлами**
    - Загрузка и хранение медиафайлов
    - Получение и скачивание файлов
    - Удаление файлов
    - Поддержка контрольных сумм (checksum) для исключения дубликатов

- 🔑 **Аутентификация и авторизация**
    - JWT (access/refresh токены)
    - Spring Security с кастомным фильтром
    - Разделение ролей пользователей

- 👤 **Работа с пользователями**
    - Регистрация
    - Подтверждение email
    - Хранение токенов обновления (refresh tokens)
    - Сброс пароля через email

- 📧 **Email-уведомления**
    - Отправка писем подтверждения
    - Отправка писем для восстановления пароля
    - Поддержка HTML-шаблонов

- 🗄 **База данных**
    - Liquibase для миграций
    - Таблицы: пользователи, токены, запросы email, медиафайлы

---

## 🛠️ Технологии

- **Java 17+**
- **Spring Boot** (Web, Security, Data JPA, Mail)
- **JWT**
- **Liquibase**
- **Gradle**
- **PostgreSQL**
- **Thymeleaf** (шаблоны писем)
- **Docker** (окружения `.env.local`, `.env.docker`)
