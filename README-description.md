# 📸 Photo Cloud Server

**Photo Cloud Server** — это серверная часть проекта для организации собственного облачного хранилища фотографий, видео и других файлов.  
Проект рассчитан на развёртывание на личном сервере и предоставляет REST API для регистрации пользователей, загрузки, хранения и скачивания файлов, организации файловой структуры.

- - проанализировать этот список, исключить в локальной БД из последующей загрузки уже существующие файлы (совпадение файлов по хэшкоду)
- - признак загрузки файла - прилетает соответствующий ответ от сервера. После этого в локальной БД меняется статус
- - статусы файлов (к примеру, можно менять):
- - - НОВЫЙ - новый обнаруженный в папке камеры файл, сразу заполняется БД и вычисляется ХЭШ (или это стоит разделить, сначала занести, со статусом НОВЫЙ. Потом высилять ХЭШ и менять статус на НА ПРОВЕРКУ?)
- - - НА ПРОВЕРКУ - кидается запрос на проверку по ХЭШ
- - - НА ПРОВЕРКЕ - отправлен запрос на сервер, пока ожидание
- - - НА ЗАГРУЗКУ - установлено, что такого файла нет, его следует загрузить на сервер
- - - ЗАГРУЖАЕТСЯ - начата загрузка файла. Может выставить какой-то таймаут? А если медленный инет, и большой файл?
Как делать загрузку - сразу или периодически?
- - - ОШИБКА - прилетела ошибка от сервера. Подумать, как тут быть? Вернуться к итерации
- - - ЗАГРУЖЕН - после подтверждения загрузки от сервера, или, если при проверке ХЭШа, он там уже есть.
- - - ЛИМИТ - получено сообщение от сервера, что лимит хранилища исчерпан. Лимит следует периодически проверять. Если, например,
там удалена часть файлов, и ещё есть место, то начать загружать по списку из БД с этим статусом, изменив на соответствующий.

## 🚀 Возможности

- 📂 **Управление файлами**
    - Загрузка и хранение медиафайлов
    - Получение и скачивание файлов
    - Удаление файлов
    - Поддержка контрольных сумм (checksum) для исключения дубликатов

- 🔑 **Аутентификация и авторизация**
    - JWT (access/refresh токены)
    - Spring Security с кастомным фильтром
    - Разделение ролей пользователей

- 👤 **Работа с пользователями**
    - Регистрация
    - Подтверждение email
    - Хранение токенов обновления (refresh tokens)
    - Сброс пароля через email

- 📧 **Email-уведомления**
    - Отправка писем подтверждения
    - Отправка писем для восстановления пароля
    - Поддержка HTML-шаблонов

- 🗄 **База данных**
    - Liquibase для миграций
    - Таблицы: пользователи, токены, запросы email, медиафайлы
